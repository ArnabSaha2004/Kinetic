# Assets Folder

This folder should contain your app's assets. For a complete Expo setup, you'll need:

## Required Assets

1. **icon.png** (1024x1024) - App icon
2. **splash.png** (1284x2778 for iPhone 12 Pro Max) - Splash screen
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon
4. **favicon.png** (48x48) - Web favicon

## Quick Setup

You can generate these assets using:

```bash
npx expo install expo-splash-screen
npx expo install @expo/cli
```

Or create simple placeholder assets:
- Use any 1024x1024 PNG for icon.png and adaptive-icon.png
- Use any high-resolution PNG for splash.png
- Use any 48x48 PNG for favicon.png
